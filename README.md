# Hangman
 Hangman game built with Python that features random word selection and a limited-attempt guessing system. This project demonstrates the use of loops, string manipulation, and user input validation.

# Overview
This is a fun Python Hangman game where you try to guess a secret word letter by letter.
You have 6 chances — guess wisely before you run out of attempts!

# How It Works
A random word is chosen from a small list.

The word is shown with blanks (_) for unguessed letters.

You type one letter at a time.

If the letter is correct → it appears in the word.

If wrong → you lose one attempt.

Win if you guess all letters before attempts run out.

Lose if attempts reach zero.

# Key Concepts
Functions → game logic in play_hangman()

Random → picks a secret word

If-elif-else → checks guesses

Loops → keeps the game running

Input/Output → player interaction
